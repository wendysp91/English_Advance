## BLOQUE A — Phrasal Verbs de alta frecuencia
*Los nativos los usan constantemente. Sin estos, sonás a libro de texto.*
*Formato: chunk completo en contexto, no el verbo solo.*

| Español | Phrasal verb en contexto |
|---|---|
| Levantarse | I usually wake up around 7 |
| Salir / irse | I get out of work at 6 / Let's get out of here |
| Subirse / abordarse | I got on the bus / Get in the car |
| Bajarse | I got off at the wrong stop |
| Regresar | I'll be back / I got back late |
| Lograrlo / pasar algo difícil | I'll get through it |
| Recoger / aprender algo | Can you pick me up? / I picked it up quickly |
| Salir (de salida social) | Do you want to go out tonight? |
| Seguir / continuar | Go on / Go ahead |
| Resultar ser | It turned out to be fine |
| Terminar siendo/haciendo | I ended up staying home |
| Rendirse | I'm not going to give up |
| Averiguar / descubrir | I need to figure this out / I found out yesterday |
| Aparecer / llegar | She showed up late |
| Seguir haciendo | Keep going / Keep it up |
| Aguantar algo molesto | I can't put up with this |
| Posponer | Let's put it off until next week |
| Encender / apagar | Turn it on / Turn it off |
| Subir / bajar volumen | Turn it up / Turn it down |
| Configurar / armar | I need to set this up |
| Salir bien / ejercitarse | It worked out / I'm going to work out |
| Pasar el rato | Let's hang out |
| Quedarse sin | I ran out of time / I'm running out of ideas |
| Toparse con alguien | I ran into her yesterday |
| Mencionar / traer un tema | He brought it up in the meeting |
| Separarse / dejar de funcionar | They broke up / My car broke down |
| Continuar | Carry on / Keep going |
| Despegar / quitarse ropa | The flight takes off at 8 / Take off your jacket |
| Sacar / llevar afuera | Can you take out the trash? |

## BLOQUE B — Patrones con modales
*No son tiempos verbales — son los patrones más usados para pedir, sugerir, especular.*

| Situación | Patrón |
|---|---|
| Pedir algo con educación | Could you...? / Would you mind...? |
| Ofrecer ayuda | Can I help you with that? / Do you want me to...? |
| Hacer una sugerencia | We could... / Why don't we...? |
| Especular sobre el presente | She must be tired / He might be working |
| Especular sobre el pasado | It must have been hard / She might have forgotten |
| Obligación fuerte | I have to... / I need to... |
| Obligación leve | I should probably... / I ought to... |
| Ausencia de obligación | You don't have to... / You don't need to... |
| Posibilidad | It might... / It could... / There's a chance that... |
| Capacidad | I can handle it / I can't do this right now |
| Pedir permiso natural | Can I...? / Is it okay if I...? |
| Hacer una oferta | Shall I...? / I'll get that for you |

## BLOQUE C — Preguntas automáticas
*Estas tienen que salir sin pensar. Son el 80% de las preguntas en conversación.*

| Situación | Pregunta |
|---|---|
| No entendiste algo | What do you mean? / Sorry, what? |
| Preguntar qué pasa | What's going on? / What's up? |
| Preguntar cómo estuvo algo | How was it? / How did it go? |
| Preguntar por tiempo | How long have you been...? / How long does it take? |
| Preguntar opinión | What do you think? / What's your take on it? |
| Preguntar experiencia | Have you ever...? / Have you tried...? |
| Preguntar planes | What are you up to? / What are you doing this weekend? |
| Preguntar causa | How come? / Why is that? |
| Confirmar que entendiste | Does that make sense? / You know what I mean? |
| Preguntar preferencia | Which one do you prefer? / What would you rather do? |
| Preguntar por la primera vez | Is this your first time...? |
| Retomar el tema | So anyway, where were we? / So, what were you saying? |
| Preguntar por alguien | How's [name] doing? |
| Preguntar por recomendación | What would you recommend? / What do you suggest? |
| Verificar información | Wait, so...? / Hold on, did you say...? |

## BLOQUE D — Patrones de tiempo verbal como chunks
*No estudies las reglas — memoriza los patrones como frases completas.*

### Present Simple — hábitos y hechos
| Español | Chunk |
|---|---|
| Siempre hago... | I always... / I never... |
| ¿Con qué frecuencia...? | How often do you...? |
| No suelo... | I don't usually... |
| ¿Qué haces para...? | What do you do to...? |
| Trabajo en... | I work in / at... |

### Present Continuous — ahora y planes cercanos
| Español | Chunk |
|---|---|
| Estoy haciendo X ahora | I'm working on something / I'm in the middle of... |
| ¿Qué estás haciendo? | What are you doing? / What are you working on? |
| Estoy intentando... | I'm trying to... |
| Está cambiando | Things are changing / It's getting better |
| Tengo planes ya hechos | I'm meeting a friend tonight / I'm going to... |

### Past Simple — contar historias
| Español | Chunk |
|---|---|
| Fue... | It was... / That was... |
| Pasó que... | So what happened was... |
| En ese momento... | At that point, I... |
| Al final resultó... | It turned out... |
| No sabía que... | I didn't know that... |
| Fue la primera vez que... | It was the first time I ever... |

### Present Perfect — experiencia y relevancia al presente
| Español | Chunk |
|---|---|
| ¿Alguna vez has...? | Have you ever...? |
| Nunca he... | I've never... |
| Ya lo hice | I've already done it |
| Todavía no | I haven't... yet |
| Hace cuánto estás... | How long have you been...? |
| He estado haciendo X | I've been doing X for / since... |
| Acabo de... | I just... / I've just... |

### Going to / Will — futuro hablado
| Español | Chunk |
|---|---|
| Planeo hacer X (ya decidido) | I'm going to... |
| Promesa / decisión espontánea | I'll do it / I'll take care of it |
| Predicción | It's going to be... / I think it'll... |
| Oferta espontánea | I'll get it / I'll help you with that |
| Amenaza amistosa | You'll see / You're going to love this |

### Conditionals — los dos que importan en conversación
| Español | Chunk |
|---|---|
| Si pasa X, hago Y (posible) | If I have time, I'll... |
| Situación hipotética | If I were you, I'd... / If I had more money, I'd... |
| ¿Qué harías si...? | What would you do if...? |
| Sería bueno si... | It would be great if... |
| En caso de que... | Just in case... / In case something comes up... |

## BLOQUE E — Negaciones naturales
*Las negaciones de libro suenan artificiales. Estas suenan a nativo.*

| Español | Negación natural |
|---|---|
| No realmente | Not really |
| Para nada | Not at all |
| Ni de cerca | Not even close |
| Ni siquiera | I didn't even... / I can't even... |
| Apenas | I barely... / I hardly... |
| Ya no | Not anymore / Not really, no |
| Tampoco | Neither do I / Me neither |
| No necesariamente | Not necessarily |
| No siempre | Not always / Not every time |
| No tan... | Not that + adjective (not that bad / not that far) |
| No me importa | I don't mind / I don't really care |
| No tengo ni idea | I have no idea / I have no clue |

## BLOQUE F — Verbos de alta frecuencia con sus patrones
*Los 10 verbos más usados en inglés conversacional. Aprende sus combinaciones, no el verbo solo.*

**GET** — el verbo más versátil del inglés hablado
| Chunk | Significado |
|---|---|
| I get it | Lo entiendo |
| I get what you mean | Entiendo lo que dices |
| I get nervous / tired / hungry | Me pongo nervioso/cansado/hambriento |
| I got your message | Recibí tu mensaje |
| Can you get me...? | ¿Me traés/conseguís...? |
| I'll get there | Voy a llegar / lo voy a lograr |
| It gets better | Mejora con el tiempo |

**MAKE** — crear, causar, forzar
| Chunk | Significado |
|---|---|
| It makes sense | Tiene sentido |
| It makes me feel... | Me hace sentir... |
| Make sure you... | Asegurate de... |
| I made a mistake | Cometí un error |
| It makes a difference | Hace la diferencia |
| I can't make it | No puedo ir / no puedo llegar |

**THINK** — opinión y proceso mental
| Chunk | Significado |
|---|---|
| I think so / I don't think so | Creo que sí / no |
| I'm thinking about... | Estoy pensando en... |
| What do you think? | ¿Qué opinás? |
| I thought it was... | Pensé que era... |
| Now that I think about it... | Ahora que lo pienso... |
| I think it depends | Creo que depende |

**KNOW** — conocimiento y certeza
| Chunk | Significado |
|---|---|
| I know, right? | ¿Verdad? / Ya sé |
| You know what I mean? | ¿Me entendés? |
| I don't know about that | No estoy seguro de eso |
| As far as I know... | Que yo sepa... |
| I had no idea | No tenía idea |
| Come to think of it... | Ahora que lo menciono... |

**FEEL** — estados y opiniones informales
| Chunk | Significado |
|---|---|
| I feel like... | Siento que... / Tengo ganas de... |
| I feel the same way | Pienso lo mismo |
| How do you feel about...? | ¿Qué pensás de...? |
| It feels like... | Se siente como... |
| I don't feel like it | No tengo ganas |
